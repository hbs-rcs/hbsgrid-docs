
# NaviCat

NaviCat is a user-friendly GUI for MariaDB. To purchase a license please
contact us at research@hbs.edu.

Once you have Navicat you will need to create a new connection to
MariaDB. Under the File menu please select New Connection- MariaDB.
Please enter the specifications for each tab according to the following
screen shots. For the General tab please note that:

    Host Name/IP Address = hostname
    User = username
    Password = password

(Please [contact RCS](mailto:research@hbs.edu) for the hostname if you
do not know it.)

![](https://hbs.edu/research-computing-services/Shared%20Documents/Grid/navicat_mariadb_1_01.png)

For the SSL tab please select Use SSL and Use Authentication. Make sure
to specify the path to your CA Certificate:

![](http://hbs.edu/research-computing-services/Shared%20Documents/Grid/navicat_mariadb_2.png)

(Please [contact RCS](mailto:research@hbs.edu) for the CA certificate if
you do not have it.)

Click Test Connection to make sure your MariaDB Connection is set up
correctly. Once the MariaDB connection is established, you should see
the connection name in the left panel as shown in the screen shot below:

![](https://hbs.edu/research-computing-services/Shared%20Documents/Grid/navicat_mariadb_3.png)

Double-click on the connection name to expand and open your database.
Please check out the detailed Navicat User Guide accessible via the Help
menu- Navicat Premium Help for a Navicat orientation. You are also
welcomed to contact us for help at <research@hbs.edu>.
