
The HBSGrid offers artificial intelligence(AI) and machine learning (ML)
capabilities through its Conda AI environment. This environment
includes:

-   Python v3.8
-   [TensorFlow](https://www.tensorflow.org/){url="https://www.tensorflow.org/"}(-gpu)
    v2.0
-   [PyTorch](https://pytorch.org/){url="https://pytorch.org/"} v1.3
-   [Keras](https://keras.io/){url="https://keras.io/"} v2.3
-   numpy v1.19
-   scipy 1.5
-   pandas v1.1
-   scikitlearn v0.23
-   matplotlib v3.3
-   the [Spyder
    IDE](https://www.spyder-ide.org/){url="https://www.spyder-ide.org/"
    target="_blank"} v4.1

The AI environment can be loaded using the following command in your
terminal window:

`module load AI/python.3.7.7`

To learn more about software modules on the HBSGrid, please see our
general info on [software
modules](../selectingsoftware/overview.md#software-modules).

The ML/AI frameworks can be run on the standard compute nodes or on our
compute node equipped with NVidia GPUs. Please see [GPU
Computing](../running-jobs/scaling-work.md#GPU)
for more information on how to submit jobs to utilize the GPUs.
